Instructions:
1. Copy each variable from this file (highlight the entire {{variableName}} including braces)
2. In Word, position cursor where you want the variable
3. Use Ctrl+Shift+V to paste as plain text
4. Verify the variable appears as one continuous selection when highlighted
5. Apply formatting to surrounding text, not to the variable itself
